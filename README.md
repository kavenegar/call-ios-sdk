## ساخت حساب کاربری


اگر در سرویس کاوه نگار عضو نیستید میتوانید از لینک 
[عضویت](http://panel.kavenegar.com/client/membership/register)
 ثبت نام و اکانت آزمایشی برای تست سرویس تماس اینترنتی دریافت نمایید.
